## Overview

Halaman website yang kamu lihat memiliki **3 aspek berbeda**:

- **Skeleton (Blade/HTML):** Seperti blok saat menulis di Notion
- **Skin (CSS):** Styling tampilan
- **Brain (PHP/Laravel):** Logic dan backend

---

# PHP

**PHP (Hypertext Preprocessor)** adalah bahasa pemrograman server-side yang digunakan untuk membangun website dinamis.
Ini adalah **bahasa utama** yang harus dipelajari sebelum masuk ke Laravel.

---

## Cara Memulai

Install **XAMPP** atau **Laragon** untuk environment lokal. Buat file bernama: `index.php`.

Editor yang direkomendasikan:

- Visual Studio Code
- Cursor
- PhpStorm

Contoh PHP dasar:

```php
<?php
echo "Hello World";

$nama = "Budi";
echo "Halo, " . $nama;
```

---

## Tipe Data & Variabel

PHP adalah bahasa yang _loosely typed_. Variabel diawali dengan tanda `$`:

```php
$nama   = "Budi";              // String
$umur   = 25;                  // Integer
$tinggi = 1.75;                // Float
$aktif  = true;                // Boolean
$hobi   = ["coding", "membaca"]; // Array
```

# Laravel

**Laravel** adalah framework PHP paling populer saat ini. Laravel menyederhanakan proses pengembangan web dengan menyediakan tools bawaan yang lengkap.

Dokumentasi resmi: https://laravel.com/docs

## Cara Memulai

Install **Composer** terlebih dahulu (package manager PHP):

```bash

composer create-project laravel/laravel nama-proyek

# Masuk ke folder proyek
cd nama-proyek

# Jalankan server development
php artisan serve
```

# Blade Templates (Pengganti HTML)

**Blade** adalah template engine bawaan Laravel. tetapi untuk PHP.
File Blade menggunakan ekstensi `.blade.php`:

```html
<!-- resources/views/welcome.blade.php -->
<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <title>{{ $judul }}</title>
  </head>
  <body>
    <h1>{{ $judul }}</h1>
    <p>Halo, {{ $nama }}!</p>
  </body>
</html>
```

---

## Komponen Blade (Pengganti React Components)

Seperti React Components, Blade juga memiliki komponen yang bisa dipakai ulang:

```html
<!-- resources/views/components/button.blade.php -->
<button {{ $attributes->merge(['class' => 'btn btn-primary']) }}> {{ $slot }}</button>

<!-- Cara pakai di view lain -->
<x-button class="btn-lg"> Login </x-button>
```

---

## Logika di Blade

```html
{{-- Kondisional --}} @if ($user->isAdmin())
<p>Selamat datang, Admin!</p>
@else
<p>Selamat datang, {{ $user->name }}</p>
@endif {{-- Looping --}} @foreach ($posts as $post)
<div>
  <h2>{{ $post->title }}</h2>
  <p>{{ $post->content }}</p>
</div>
@endforeach
```

# Routing

Di Laravel, routing didefinisikan di file `routes/web.php`:

```php
// routes/web.php
Route::get('/', function () {
    return view('welcome');
});

Route::get('/user', [UserController::class, 'index']);
Route::get('/user/{id}', [UserController::class, 'show']);
Route::post('/user', [UserController::class, 'store']);
Route::put('/user/{id}', [UserController::class, 'update']);
Route::delete('/user/{id}', [UserController::class, 'destroy']);
```

---

## Dynamic Routes

```php
// Definisi route dengan parameter
Route::get('/user/{id}', [UserController::class, 'show']);

// Di dalam Controller
public function show($id)
{
    $user = User::findOrFail($id);
    return view('users.show', compact('user'));
}
```

---

# Controller

Controller di Laravel:

```php
// app/Http/Controllers/UserController.php
<?php
namespace App\Http\Controllers;

use App\Models\User;

class UserController extends Controller
{
    public function index()
    {
        $users = User::all();
        return view('users.index', compact('users'));
    }

    public function show($id)
    {
        $user = User::findOrFail($id);
        return view('users.show', compact('user'));
    }
}
```

Buat Controller baru dengan Artisan:

```bash
php artisan make:controller UserController
```

# Eloquent ORM

**Eloquent** adalah ORM bawaan Laravel:

```php
// app/Models/User.php
<?php
namespace App\Models;

use Illuminate\Database\Eloquent\Model;

class User extends Model
{
    protected $fillable = ['name', 'email', 'password'];

    // Relasi: User punya banyak Post
    public function posts()
    {
        return $this->hasMany(Post::class);
    }
}
```

---

## Query dengan Eloquent

```php
// Ambil semua data
$users = User::all();

// Cari berdasarkan ID
$user = User::find(1);

// Filter
$admins = User::where('role', 'admin')->get();

// Buat data baru
User::create(['name' => 'Budi', 'email' => 'budi@email.com']);

// Update
$user->update(['name' => 'Siti']);

// Hapus
$user->delete();
```

# Migrations (Database Schema)

Migration di Laravel menggantikan **Prisma Schema**. Digunakan untuk membuat dan mengubah struktur tabel database:

```bash
# Buat migration baru
php artisan make:migration create_posts_table

# Jalankan migration
php artisan migrate

# Rollback
php artisan migrate:rollback
```

```php
// database/migrations/xxxx_create_posts_table.php
public function up(): void
{
    Schema::create('posts', function (Blueprint $table) {
        $table->id();
        $table->string('title');
        $table->text('content');
        $table->foreignId('user_id')->constrained();
        $table->timestamps();
    });
}
```

# Authentication

Laravel menyediakan **Breeze** atau **Jetstream** sebagai starter kit autentikasi, menggantikan Better Auth:

```bash
# Install Laravel Breeze (simple)
composer require laravel/breeze --dev
php artisan breeze:install

# Atau Jetstream (lengkap dengan teams)
composer require laravel/jetstream
php artisan jetstream:install livewire
```

Fitur yang sudah tersedia setelah install:

- Login dengan email/password
- Register akun baru
- Forgot password / reset password
- Social login via Laravel Socialite (Google, GitHub, dll)

# Middleware (Proteksi Route)

Middleware di Laravel berfungsi sama seperti middleware di Next.js, untuk memproteksi route:

```php
// Proteksi route dengan middleware auth
Route::get('/dashboard', function () {
    return view('dashboard');
})->middleware('auth');

// Grup route yang dilindungi
Route::middleware(['auth'])->group(function () {
    Route::get('/profile', [ProfileController::class, 'show']);
    Route::get('/settings', [SettingController::class, 'index']);
});
```

# API dengan Laravel

Laravel bisa digunakan sebagai pure API backend. Gunakan `routes/api.php` untuk mendefinisikan endpoint:

```php
// routes/api.php
Route::apiResource('users', UserController::class);

// Menghasilkan route otomatis:
// GET    /api/users          → index()
// POST   /api/users          → store()
// GET    /api/users/{id}     → show()
// PUT    /api/users/{id}     → update()
// DELETE /api/users/{id}     → destroy()
```

# Rendering di Laravel

## Server Side Rendering (Default Laravel)

Laravel secara default menggunakan SSR. Controller mengambil data lalu mengirim ke view Blade:

```php
public function index()
{
    $posts = Post::with('user')->latest()->get();
    return view('posts.index', compact('posts')); // Data dikirim ke Blade
}
```

---

## Livewire (Pengganti useState & CSR)

**Livewire** adalah tool untuk interaktivitas tanpa JavaScript:

```bash
php artisan make:livewire Counter
```

```php
// app/Livewire/Counter.php
class Counter extends Component
{
    public int $count = 0;

    public function increment()
    {
        $this->count++;
    }

    public function render()
    {
        return view('livewire.counter');
    }
}
```

```html
<!-- resources/views/livewire/counter.blade.php -->
<div>
  <h1>{{ $count }}</h1>
  <button wire:click="increment">Tambah</button>
</div>
```
